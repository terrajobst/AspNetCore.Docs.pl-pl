## <a name="conventions"></a><span data-ttu-id="20479-329">Konwencje</span><span class="sxs-lookup"><span data-stu-id="20479-329">Conventions</span></span>

<span data-ttu-id="20479-330">Ilość kodu, trzeba było pisać w kolejności Entity Framework można było tworzenie pełną bazy danych jest minimalny z powodu *konwencje*, lub założenia, które czynią Entity Framework.</span><span class="sxs-lookup"><span data-stu-id="20479-330">The amount of code you had to write in order for Entity Framework to be able to create a complete database for you is minimal because of *conventions*, or assumptions that Entity Framework makes.</span></span> <span data-ttu-id="20479-331">Niektóre z nich zostały już zanotowane lub zostały użyte bez Twojej wiedziały z nich:</span><span class="sxs-lookup"><span data-stu-id="20479-331">Some of them have already been noted or were used without your being aware of them:</span></span>

- <span data-ttu-id="20479-332">Pluralized formy nazwy klas jednostek są używane jako nazwy tabeli.</span><span class="sxs-lookup"><span data-stu-id="20479-332">The pluralized forms of entity class names are used as table names.</span></span>
- <span data-ttu-id="20479-333">Nazwy właściwości jednostki są używane dla nazw kolumn.</span><span class="sxs-lookup"><span data-stu-id="20479-333">Entity property names are used for column names.</span></span>
- <span data-ttu-id="20479-334">Właściwości jednostki, które są nazwane `ID` lub *classname* `ID` są rozpoznawane jako właściwości klucza podstawowego.</span><span class="sxs-lookup"><span data-stu-id="20479-334">Entity properties that are named `ID` or *classname* `ID` are recognized as primary key properties.</span></span>
- <span data-ttu-id="20479-335">Właściwość jest interpretowany jako właściwość klucza obcego, jeśli jest on nazwany *&lt;nazwy właściwości nawigacji&gt;&lt;nazwa właściwość klucza podstawowego&gt;* (na przykład `StudentID` dla `Student` właściwość nawigacji od `Student` jest klucz podstawowy jednostki `ID`).</span><span class="sxs-lookup"><span data-stu-id="20479-335">A property is interpreted as a foreign key property if it's named *&lt;navigation property name&gt;&lt;primary key property name&gt;* (for example, `StudentID` for the `Student` navigation property since the `Student` entity's primary key is `ID`).</span></span> <span data-ttu-id="20479-336">Właściwości klucza obcego może również mieć taką samą nazwę po prostu &lt;nazwa właściwość klucza podstawowego&gt; (na przykład `EnrollmentID` ponieważ `Enrollment` jest klucz podstawowy jednostki `EnrollmentID`).</span><span class="sxs-lookup"><span data-stu-id="20479-336">Foreign key properties can also be named the same simply &lt;primary key property name&gt; (for example, `EnrollmentID` since the `Enrollment` entity's primary key is `EnrollmentID`).</span></span>
